Live reloading


It's fast. Impatient mode updates the browser as you type.

Saves time. Instead of switch windows and refreshing browers, allows you to focus on coding.

Catch mistakes faster. See errors as you type.

Run

It only takes 3 simple steps to start using.

  1. First, make sure httpd is running. This is done by running httpd-start (M-x).
  2. Next, add file to impatient-mode, by running impatient-mode
  3. Finally, open localhost:8080/imp in your favorite browser
